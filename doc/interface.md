				- 3.1.4.1 显示搜索结果
					- 3.1.4.1.1 搜索结果为个人信息列表，用户点击列表的某个 item 则进入“个人信息详情板块”

						- 有两个文本框，用来填写年份最小值和年份最大值

							- 没有填：表示的是对年份不做筛选
							- 填最小值框A：表示筛选入学年份在最小年份A之后的人。
							- 填最大值框B：表示筛选入学年份在最大值年份B之前的人。
							- A和B都填写：表示筛选入学年份在A和B之间的人
					- 3.1.4.3.3 筛选院系
						- 默认展示一行，点击三角号展示下拉窗口，展示所有院系
